# Polymons

Polymons is a place to build games, publish them, play games made by other
people, and hang out with friends.

## Status

Polymons is in early development. The current build is a static preview of the
main website with mock games, profiles, friends, and creator pages.

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
```
